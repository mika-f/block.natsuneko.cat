# uBlacklist Subscription for Developers

uBlackList Subscriptions for Developers.  
This repository provides filters for uBlacklist for each of the categories below from `contents/blocks.json`.

| Category    | Description                                                                                                                                                                                   | URL                                                 |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| All         | A subscription containing all of the following categories                                                                                                                                     | https://block.natsuneko.cat/all.txt                 |
| Affiliate   | A website that pretends to be a technical article and tries to guide you to company's products or sign a contract                                                                             | https://block.natsuneko.cat/affiliate.txt           |
| Curation    | A website that simply summarizes articles from the primary source without adding extra information                                                                                            | https://block.natsuneko.cat/curation.txt            |
| Translation | A website that mechanically translates the articles of the primary source, or artificially translates and reprints without adding extra information                                           | https://block.natsuneko.cat/machine-translation.txt |
| Meta        | A website that make some kind of comments on articles from primary sources, expect for some large online boards                                                                               | https://block.natsuneko.cat/meta.txt                |
| News        | A website that distribute hoaxes, fake news, scientifically unverified news, politically biases news, or news that strongly incites hatred towards specific users                             | https://block.natsuneko.cat/news.txt                |
| Username    | A list of usernames that are misleading, fake news, scientifically unverified, politically biased, or significantly inciting hatred of specific users, exaggerated expressions, reprints, etc | https://block.natsuneko.cat/username.txt            |

## Add/Remove a website / username

### Fork

1. Fork this repository
2. Edit `contents/blocks.json`
3. Commit your changes
4. Open the pull request

### Issue

1. Create a issue on this repository

# 開発者向けの uBlacklist 購読リスト

開発者向けの uBlacklist 購読リストです。  
このリポジトリは、 `contents/blocks.json` から、以下のカテゴリーに応じた uBlacklist のフィルターを生成、配信しています。

| Category    | Description                                                                                                                                                                | URL                                                 |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| All         | 下記カテゴリーのすべてが含まれたリスト                                                                                                                                     | https://block.natsuneko.cat/all.txt                 |
| Affiliate   | 技術記事を装って自社製品への誘導を行ったり、契約を行わせようとしているウェブサイト                                                                                         | https://block.natsuneko.cat/affiliate.txt           |
| Curation    | 一次ソースの情報に新たな情報を付け足すこと無く、まとめただけのウェブサイト                                                                                                 | https://block.natsuneko.cat/curation.txt            |
| Translation | 一次ソースの情報を機械的に翻訳、もしくは何らかの情報を付け足すこと無く、人為的に翻訳をして転載しているだけのウェブサイト                                                   | https://block.natsuneko.cat/machine-translation.txt |
| Meta        | 一次ソースの情報に対して、何らかのコメントを行ったりするウェブサイト、ただし、一部の巨大掲示板は除く                                                                       | https://block.natsuneko.cat/meta.txt                |
| News        | デマやフェイクニュース、科学的に検証されていない、政治的に著しく偏っている、または特定ユーザーのヘイトを著しく煽っているウェブサイトのリスト                               | https://block.natsuneko.cat/news.txt                |
| Username    | デマやフェイクニュース、科学的に検証されていない、政治的に著しく偏っている、または特定ユーザーのヘイトを著しく煽っている、誇張表現、転載などを行っているユーザー名のリスト | https://block.natsuneko.cat/username.txt            |

## ウェブサイトやユーザー名の追加・削除方法

2. `contents/blocks.json` を編集
3. 変更をコミット
4. Pull Request を作成

### Issue

1. Issue を作成
